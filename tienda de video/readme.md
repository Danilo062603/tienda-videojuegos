# 🎮 GameZone

Aplicación web simple para gestionar partidas de videojuegos, registrar sesiones y visualizar progreso.

## 🚀 Características

* Registro de partidas
* Lista de juegos jugados
* Estados de partidas (En progreso / Completado)
* Interfaz moderna estilo gaming
* Diseño responsive con CSS Grid

## 🛠️ Tecnologías

* HTML5
* CSS3

## 📂 Estructura del proyecto

```
/gamezone
│── index.html
│── styles.css
│── README.md
```

## 🎯 Uso

1. Clona el repositorio:

```
git clone https://github.com/tu-usuario/gamezone.git
```

2. Abre el archivo:

```
index.html
```
